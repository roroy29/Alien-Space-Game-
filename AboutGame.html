About My Game

This game was designed by me, Roshan Roy, for a university assignment. I wrote most of the code for this game and also used some code from websites that I referenced. These websites are commented on my jscript.js file and I will also reference these below.

Game Objects

  • Player - The blue alien in the red ufo.
  • Arena - The red box in which all the aliens contain.
  • Obstacles - The 12 green aliens in the pink ufo.
  • Points - The pink ufos touch the bottom red bar you get +1 points.
